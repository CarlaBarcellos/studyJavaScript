## Setup
Instale as dependencias.
```terminal
npm init
npm i
```

Para adicionar dependencisas
```terminal
npm NOME DO MODULO -S
```

Reload Automaticly (Server)
```terminal
sudo npm install -g nodemon
nodemon app.js
```

Live Reload (View)
```terminal
express-livereload
```
